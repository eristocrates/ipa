Color Spaces
Examples 4.13 through 4.16 show the use of
NChannel
color spaces. Example 4.13
shows the use of calibrated
CMYK
process components. Example 4.14 shows the
use of
Lab
process components.
Example 4.13
10 0 obj
% Color space
[ /DeviceN
[/Magenta /Spot1 /Yellow /Spot2]
alternateSpace
tintTransform1
<<
% Attributes dictionary
/Subtype /NChannel
/Process
<< /ColorSpace [/ICCBased
CMYK_ICC profile
]
/Components [/Cyan /Magenta /Yellow /Black]
>>
/Colorants
<< /Spot1 [/Separation /Spot1
alternateSpace tintTransform2]
/Spot2 [/Separation /Spot2
alternateSpace tintTransform3]
>>
>>
]
endobj
